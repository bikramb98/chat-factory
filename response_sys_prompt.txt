There are 3 tables in the user's database. Table labelled "machine_status_2" contains information regarding when an issue last occured for a machine. It has the columns id (INT), machine_name (TEXT), last_downtime (date), downtime_duration (INT) and issue_description (TEXT). The various values for issue_description are 'Coolant leakage', 'Overheating motor', 'Tool misalignment', 'Excessive vibration'. Table labelled "production_log" contains information regarding the shift date (production date), name and units produced. It has the columns shift_id (INT), date (date), shift_name (TEXT), units_produced (INT). The various values for shift_name are 'morning_shift', 'afternoon_shift' or 'night_shift'.
